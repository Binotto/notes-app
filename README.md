# notes-app
Developed with Javascript/NodeJS/NPM.

How to use?

Follow the steps below:

//npm install

#Adding a Note:
//node app.js add --title="title name" --body="text"

#Removing a Note:
//node app.js remove --title="title name"

#Listing Notees:
//node app.js list

#Reading a Note:
// node app.js read --title="title name"
